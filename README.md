# ARC

ARC는 AI가 연작 단편소설을 작성하는 취미 프로젝트다.

입력:

- 최소 세계관
- 에피소드 한 줄 아이디어

과정:

- 계획
- 초안
- 비판적 리뷰 1회
- 수정 1회
- 최종본

출력:

- 완성된 단편소설 final.md

이번 초기 버전에서 하지 않는 것:

- 복잡한 승인 게이트
- 범용 플랫폼
- 여러 모델 투표
- 무한 수정

첫 목표:

실제 단편소설 한 편을 ARC가 처음부터 끝까지 완성한다.

이번에는 코드부터 만들지 않는다. 이 README를 정본으로 승인한 뒤, 필요한 최소 기능만 한 번에 설계한다.
